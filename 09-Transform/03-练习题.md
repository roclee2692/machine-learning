## 答案提示

### 练习 1 答案

```
Q·K^T = [[1, 0], [0, 1]]   ← 单位矩阵
scores = Q·K^T / √2 = [[0.707, 0], [0, 0.707]]
weights = softmax(scores) ≈ [[0.67, 0.33], [0.33, 0.67]]
output = weights · V ≈ [[1.66, 2.66], [2.34, 3.34]]
```

含义：当 Q=K=I 时，每个位置主要关注自己（权重 ~0.67），也稍微关注另一个位置（权重 ~0.33）。

### 练习 2 答案

1. Q·K^T 每个元素 = Σ(Q_i · K_i)，i=1..64。均值=0，方差=64。
2. 不除以 √64=8，softmax 输入值范围 ±30，输出接近 one-hot → 梯度消失。
3. softmax([0.1, 0.2, 0.3]) ≈ [0.30, 0.33, 0.37]（平滑）
   softmax([10, 20, 30]) ≈ [0.00, 0.00, 1.00]（尖锐）

### 练习 5 答案

```python
def create_causal_mask(seq_len):
    mask = torch.tril(torch.ones(seq_len, seq_len))  # 下三角
    return mask.unsqueeze(0).unsqueeze(0)  # (1, 1, seq_len, seq_len)
```

### 练习 8 答案要点

1. BERT 用 Encoder：因为 BERT 做完形填空（Masked LM），需要双向上下文。
2. GPT 用 Decoder：因为 GPT 做下一个词预测，必须用 Causal Mask 防止看到答案。
3. 如果 BERT 用在生成任务：它能看到所有位置，"下一个词预测"就变成了"复制答案"。
4. Encoder-Decoder：Encoder 编码整个输入，Decoder 逐词生成 + 通过 Cross-Attention 参考输入。
