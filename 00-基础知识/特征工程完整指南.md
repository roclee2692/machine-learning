## 1. 基础概念

### 1.1 什么是特征？

**特征 (Feature)** = 用于描述样本的属性/变量

```python
# 一个房价预测的例子
样本1: [面积=90, 卧室=2, 楼层=5, 房龄=3] → 房价=300万
       ↑_______________特征_______________↑   ↑____标签____↑
```

**专业术语对照：**

- **特征 (Feature)** = 属性 (Attribute) = 变量 (Variable)
- **标签 (Label)** = 目标值 (Target) = 因变量 (Dependent Variable)
- **样本 (Sample)** = 实例 (Instance) = 数据点 (Data Point)

### 1.2 三个核心概念

| 概念 | 英文    | 说明     | 例子               |
| ---- | ------- | -------- | ------------------ |
| 特征 | Feature | 输入变量 | 面积、房龄         |
| 标签 | Label   | 输出变量 | 房价               |
| 样本 | Sample  | 一条数据 | 一套房子的所有信息 |

### 1.3 特征类型

#### 数值型特征 (Numerical)

- **连续型**：身高(175.5cm)、温度(36.5°C)、价格
- **离散型**：年龄(25岁)、数量(3个)、评分(1-5星)

#### 类别型特征 (Categorical)

- **无序类别**：颜色(红/蓝/绿)、性别(男/女)、城市名
- **有序类别**：学历(小学<初中<高中<大学)、等级(低/中/高)

#### 时间型特征 (Temporal)

- 日期、时间戳、年份

#### 文本型特征 (Text)

- 评论、描述、文章

## 2. 特征预处理

### 2.1 缺失值处理

#### 方法1：删除

```python
# 删除含缺失值的行
df.dropna()

# 删除缺失值过多的列（如超过30%）
threshold = 0.3
df = df.loc[:, df.isnull().mean() < threshold]
```

**适用场景**：

- ✅ 缺失数据很少（<5%）
- ✅ 数据量充足
- ❌ 缺失不是随机的（可能丢失重要模式）

#### 方法2：填充

**统计值填充**

```python
# 均值填充（数值型）
df['age'].fillna(df['age'].mean(), inplace=True)

# 中位数填充（更稳健，不受异常值影响）
df['income'].fillna(df['income'].median(), inplace=True)

# 众数填充（类别型）
df['city'].fillna(df['city'].mode()[0], inplace=True)

# 前向/后向填充（时间序列）
df.fillna(method='ffill')  # 用前一个值填充
df.fillna(method='bfill')  # 用后一个值填充
```

**高级填充**

```python
from sklearn.impute import SimpleImputer, KNNImputer

# 基础填充器
imputer = SimpleImputer(strategy='mean')  # 'median', 'most_frequent', 'constant'
df_filled = imputer.fit_transform(df)

# KNN填充（用相似样本的值）
imputer = KNNImputer(n_neighbors=5)
df_filled = imputer.fit_transform(df)
```

#### 方法3：预测填充

```python
# 用其他特征预测缺失值
from sklearn.linear_model import LinearRegression

# 把缺失值当作预测目标
known = df[df['age'].notnull()]
unknown = df[df['age'].isnull()]

model = LinearRegression()
model.fit(known[['income', 'education']], known['age'])
df.loc[df['age'].isnull(), 'age'] = model.predict(unknown[['income', 'education']])
```

### 2.2 异常值处理

#### 检测方法

**1. 统计方法（3σ原则）**

```python
# 计算上下界
mean = df['price'].mean()
std = df['price'].std()
lower = mean - 3*std
upper = mean + 3*std

# 标记异常值
outliers = (df['price'] < lower) | (df['price'] > upper)
print(f"异常值数量: {outliers.sum()}")
```

**2. IQR方法（四分位距）**

```python
Q1 = df['price'].quantile(0.25)
Q3 = df['price'].quantile(0.75)
IQR = Q3 - Q1

lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

outliers = (df['price'] < lower) | (df['price'] > upper)
```

**3. 可视化检测**

```python
import matplotlib.pyplot as plt

# 箱线图
plt.boxplot(df['price'])
plt.show()

# 散点图
plt.scatter(df['area'], df['price'])
plt.show()
```

#### 处理方法

```python
# 方法1：删除
df = df[~outliers]

# 方法2：截断（Capping）
df['price'] = df['price'].clip(lower=lower, upper=upper)

# 方法3：变换（对数、平方根）
df['price_log'] = np.log1p(df['price'])

# 方法4：分箱（后面会讲）
```

### 2.3 特征缩放

#### 为什么需要缩放？

```
特征1（面积）: [50, 60, 70, ..., 200]  范围: 150
特征2（楼层）: [1, 2, 3, ..., 30]     范围: 29
```

**问题**：范围差异大的特征会主导模型训练

**需要缩放的算法**：

- ✅ 线性回归（梯度下降版本）
- ✅ 逻辑回归
- ✅ K近邻 (KNN)
- ✅ K-Means聚类
- ✅ SVM
- ✅ 神经网络

**不需要缩放的算法**：

- ❌ 决策树
- ❌ 随机森林
- ❌ XGBoost

#### 归一化 (Min-Max Scaling)

**公式**：

```
x' = (x - x_min) / (x_max - x_min)
```

**结果**：映射到 [0, 1]

```python
from sklearn.preprocessing import MinMaxScaler

scaler = MinMaxScaler()
df[['area', 'price']] = scaler.fit_transform(df[['area', 'price']])

# 或指定范围
scaler = MinMaxScaler(feature_range=(-1, 1))
```

**优点**：

- 保留原始分布形状
- 适合有明确上下界的数据

**缺点**：

- 对异常值敏感

#### 标准化 (Standardization)

**公式**：

```
x' = (x - μ) / σ
```

其中 μ 是均值，σ 是标准差

**结果**：均值=0，标准差=1

```python
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
df[['area', 'price']] = scaler.fit_transform(df[['area', 'price']])
```

**优点**：

- 对异常值较稳健
- 适合正态分布数据
- 更常用

**缺点**：

- 没有固定范围

#### 选择建议

| 场景                          | 推荐方法       |
| ----------------------------- | -------------- |
| 数据有明确边界（如图像0-255） | MinMaxScaler   |
| 数据接近正态分布              | StandardScaler |
| 有异常值                      | RobustScaler   |
| 默认选择                      | StandardScaler |

#### RobustScaler（稳健缩放）

```python
from sklearn.preprocessing import RobustScaler

# 使用中位数和IQR，对异常值更稳健
scaler = RobustScaler()
df_scaled = scaler.fit_transform(df)
```

### 2.4 类别特征编码

#### One-Hot编码（独热编码）

**原理**：每个类别变成一列二进制特征

```python
# 原始数据
colors = ['红', '蓝', '绿', '红', '蓝']

# One-Hot编码后
# 红  蓝  绿
# 1   0   0
# 0   1   0
# 0   0   1
# 1   0   0
# 0   1   0
```

```python
import pandas as pd

# 方法1：pandas
df_encoded = pd.get_dummies(df, columns=['color', 'city'])

# 方法2：sklearn
from sklearn.preprocessing import OneHotEncoder

encoder = OneHotEncoder(sparse=False, drop='first')  # drop='first'避免多重共线性
encoded = encoder.fit_transform(df[['color']])
```

**适用**：

- ✅ 无序类别特征
- ✅ 类别数量较少（<10个）

**缺点**：

- ❌ 类别太多会导致维度爆炸

#### Label编码（标签编码）

```python
# 原始：['低', '中', '高', '低', '中']
# 编码后：[0, 1, 2, 0, 1]

from sklearn.preprocessing import LabelEncoder

encoder = LabelEncoder()
df['level_encoded'] = encoder.fit_transform(df['level'])
```

**适用**：

- ✅ 有序类别（低<中<高）
- ✅ 树模型（决策树、随机森林）

**注意**：

- ❌ 不要用于无序类别+线性模型（会引入虚假的大小关系）

#### Ordinal编码（序号编码）

```python
from sklearn.preprocessing import OrdinalEncoder

# 手动指定顺序
encoder = OrdinalEncoder(categories=[['小学', '初中', '高中', '大学', '研究生']])
df['education_encoded'] = encoder.fit_transform(df[['education']])
```

#### Target编码（目标编码）

```python
# 用该类别对应的目标值均值替换

# 例如：城市 → 该城市的平均房价
city_means = df.groupby('city')['price'].mean()
df['city_encoded'] = df['city'].map(city_means)
```

**优点**：

- 类别数量很多时仍有效
- 包含目标信息

**缺点**：

- 容易过拟合
- 需要交叉验证配合使用

## 3. 特征构造

### 3.1 多项式特征

```python
from sklearn.preprocessing import PolynomialFeatures

# 原始特征: [x1, x2]
# 2次多项式: [1, x1, x2, x1², x1·x2, x2²]

poly = PolynomialFeatures(degree=2, include_bias=False)
X_poly = poly.fit_transform(X)
```

**应用**：

- 捕捉非线性关系
- 特征交互

### 3.2 交互特征

```python
# 手动创建
df['area_per_room'] = df['area'] / df['rooms']  # 每个房间的面积
df['price_per_sqm'] = df['price'] / df['area']   # 每平米价格

# 数学运算
df['feature_sum'] = df['f1'] + df['f2']
df['feature_product'] = df['f1'] * df['f2']
df['feature_ratio'] = df['f1'] / (df['f2'] + 1)  # 避免除0
```

### 3.3 时间特征

```python
df['date'] = pd.to_datetime(df['date'])

# 提取组件
df['year'] = df['date'].dt.year
df['month'] = df['date'].dt.month

df['hour'] = df['date'].dt.hour

# 派生特征

df['is_holiday'] = df['date'].isin(holidays).astype(int)
df['quarter'] = df['date'].dt.quarter

# 时间差
```

### 3.4 文本特征

```python
from sklearn.feature_extraction.text import CountVectorizer, TfidfVectorizer

# 词袋模型
vectorizer = CountVectorizer(max_features=1000)
X_counts = vectorizer.fit_transform(df['text'])

# TF-IDF
tfidf = TfidfVectorizer(max_features=1000, ngram_range=(1, 2))
X_tfidf = tfidf.fit_transform(df['text'])

# 简单统计特征
df['text_length'] = df['text'].str.len()

df['capital_count'] = df['text'].str.count(r'[A-Z]')
```

### 3.5 分箱 (Binning)

```python
# 等宽分箱
df['age_binned'] = pd.cut(df['age'], bins=5, labels=['很年轻', '年轻', '中年', '中老年', '老年'])

# 等频分箱（每个箱样本数相同）
df['income_binned'] = pd.qcut(df['income'], q=4, labels=['低', '中', '中高', '高'])

# 自定义边界
bins = [0, 18, 35, 60, 100]
labels = ['未成年', '青年', '中年', '老年']
df['age_group'] = pd.cut(df['age'], bins=bins, labels=labels)
```

**优点**：

- 减少异常值影响
- 捕捉非线性关系
- 增强鲁棒性

## 4. 特征选择

### 4.1 为什么要选择特征？

- ✅ 减少过拟合
- ✅ 提高训练速度
- ✅ 提高模型可解释性
- ✅ 减少存储需求

### 4.2 过滤法 (Filter)

#### 方差阈值

```python
from sklearn.feature_selection import VarianceThreshold

# 删除方差<0.1的特征
selector = VarianceThreshold(threshold=0.1)
X_selected = selector.fit_transform(X)
```

#### 相关系数

```python
# 特征与目标的相关性
correlation = df.corr()['target'].abs().sort_values(ascending=False)
print(correlation.head(10))

# 选择相关性>0.3的特征
selected_features = correlation[correlation > 0.3].index.tolist()
```

#### 卡方检验

```python
from sklearn.feature_selection import SelectKBest, chi2

# 选择最好的K个特征
selector = SelectKBest(chi2, k=10)
X_selected = selector.fit_transform(X, y)

# 查看得分
scores = pd.DataFrame({
    'Feature': feature_names,
    'Score': selector.scores_
}).sort_values('Score', ascending=False)
```

### 4.3 包裹法 (Wrapper)

#### 递归特征消除 (RFE)

```python
from sklearn.feature_selection import RFE
from sklearn.ensemble import RandomForestClassifier

# 选择最重要的10个特征
model = RandomForestClassifier()
rfe = RFE(model, n_features_to_select=10)
rfe.fit(X, y)

# 查看选中的特征
selected = [f for f, s in zip(feature_names, rfe.support_) if s]
print(selected)
```

### 4.4 嵌入法 (Embedded)

#### L1正则化

```python
from sklearn.linear_model import Lasso

# L1会让一些特征系数变为0
lasso = Lasso(alpha=0.1)
lasso.fit(X, y)

# 选择非零系数的特征
selected = [f for f, c in zip(feature_names, lasso.coef_) if abs(c) > 0.01]
```

#### 树模型特征重要性

```python
from sklearn.ensemble import RandomForestRegressor

rf = RandomForestRegressor()
rf.fit(X, y)

# 特征重要性
importances = pd.DataFrame({
    'Feature': feature_names,
    'Importance': rf.feature_importances_
}).sort_values('Importance', ascending=False)

print(importances.head(10))

# 选择重要性>阈值的特征
threshold = 0.01
selected = importances[importances['Importance'] > threshold]['Feature'].tolist()
```

## 6. 数据集划分

### 6.1 训练集 / 测试集

```python
from sklearn.model_selection import train_test_split

# 80-20划分
X_train, X_test, y_train, y_test = train_test_split(
```

### 6.2 训练集 / 验证集 / 测试集


### 6.3 交叉验证

```python
from sklearn.model_selection import cross_val_score, KFold

# 5折交叉验证
scores = cross_val_score(model, X, y, cv=5, scoring='accuracy')
print(f"平均分数: {scores.mean():.4f} (+/- {scores.std():.4f})")

# 自定义折数
kf = KFold(n_splits=10, shuffle=True, random_state=42)
scores = cross_val_score(model, X, y, cv=kf)
```

### 6.4 分层采样

```python
# 保持类别比例
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, stratify=y, random_state=42
)
```
