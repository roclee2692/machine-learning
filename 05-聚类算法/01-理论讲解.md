# 聚类算法 (Clustering)

## 1. 什么是聚类？

聚类是**无监督学习**，没有标签。目标是将相似的样本分组在一起，不同的样本分开。

### 1.1 聚类 vs 分类

| 特点   | 分类     | 聚类       |
| ------ | -------- | ---------- |
| 监督性 | 有监督   | 无监督     |
| 标签   | 有       | 无         |
| 目标   | 预测标签 | 发现结构   |
| 评估   | 准确率等 | 肘部法则等 |

### 1.2 应用场景

- 客户细分
- 图像分割
- 异常检测
- 推荐系统
- 搜索引擎



超参数 =  **算法的"旋钮"** ，训练前你要先把旋钮拧到合适的位置，算法才能正常工作。调超参数的过程就叫 **调参 (hyperparameter tuning)** 。

## 2. K-Means聚类

### 2.1 算法流程

1. **初始化**：随机选择K个点作为初始中心
2. **分配**：每个样本分配到最近的中心
3. **更新**：计算每个簇的新中心（均值）
4. **重复**：直到中心不再改变

### 2.2 目标函数

```
最小化：J = Σ Σ ||xᵢ - μⱼ||²

其中：
- xᵢ 是样本
- μⱼ 是第j个簇的中心
- 求和遍历所有样本和所有簇
```

### 2.3 如何选择K？

**肘部法则 (Elbow Method)**：

- 计算不同K值下的惯性 (inertia)
- 绘制K vs 惯性的曲线
- 在"肘部"处，增加K得不到更多改善

**轮廓系数 (Silhouette Coefficient)**：

```
范围 [-1, 1]
- 接近1：聚类好
- 接近0：边界样本
- 接近-1：聚类差
```

### 2.4 超参数

- **n_clusters**：簇的数量
- **init**：初始化方法（'k-means++'更好）
- **n_init**：运行次数
- **max_iter**：最大迭代次数

### 2.5 优缺点

**优点**

- ✅ 算法简单快速
- ✅ 可扩展性好
- ✅ 易于理解

**缺点**

- ❌ 需要提前指定K
- ❌ 对初始化敏感
- ❌ 对球形簇假设强
- ❌ 对异常值敏感
- ❌ 不适应不同密度的簇

## 3. 层次聚类 (Hierarchical Clustering)

### 3.1 两种方法

**凝聚（自下而上）**：

- 开始：每个样本一个簇
- 重复：合并最接近的两个簇
- 结果：树状结构（树状图）

**分裂（自上而下）**：

- 开始：所有样本一个簇
- 重复：分裂最大的簇
- 结果：树状结构

### 3.2 链接方法

**单链接**：使用两个簇间最小距离
**完全链接**：使用两个簇间最大距离
**平均链接**：使用两个簇间平均距离
**Ward链接**：使用方差最小化

### 3.3 优缺点

**优点**

- ✅ 无需指定簇数
- ✅ 产生树状结构，信息丰富
- ✅ 灵活

**缺点**

- ❌ 计算慢 O(n²)
- ❌ 一旦决定无法撤销
- ❌ 对异常值敏感

## 4. DBSCAN聚类

### 4.1 基于密度的聚类

不假设球形簇，可以聚类任意形状的簇。

### 4.2 核心概念

**eps（ε）**：邻域半径
**min_pts**：邻域内的最小点数

**分类**：

- **核心点**：邻域内至少有min_pts个点
- **边界点**：不是核心点，但在某个核心点邻域内
- **异常点**：既不是核心点也不是边界点

### 4.3 算法流程

1. 对每个未访问的点
2. 如果是核心点，形成一个新簇
3. 递归地添加密度相连的点

### 4.4 超参数

- **eps**：邻域半径

  - 太小：大多数点被视为异常
  - 太大：簇变小
- **min_samples**：最小点数

  - 默认值 = 维度数

### 4.5 优缺点

**优点**

- ✅ 无需指定簇数
- ✅ 可聚类任意形状
- ✅ 识别异常点
- ✅ 对异常值鲁棒

**缺点**

- ❌ 超参数选择困难
- ❌ 不适应变密度簇
- ❌ 高维空间性能差

## 5. 聚类评估

### 5.1 外部评估（有标签时）

**轮廓系数**：

```
S = (b - a) / max(a, b)
```

- a：样本到同簇其他点的平均距离
- b：样本到最近不同簇的平均距离

**同质性 (Homogeneity)**：簇内样本标签相同的程度
**完整性 (Completeness)**：相同标签样本在同一簇的程度
**V-measure**：同质性和完整性的调和平均

### 5.2 内部评估（无标签时）

**惯性 (Inertia)**：簇内方差和

```
J = Σ Σ ||xᵢ - μⱼ||²
```

**轮廓系数**：-1 到 1，越大越好

**Davies-Bouldin指数**：簇间距离/簇内距离，越小越好

## 6. 三种聚类方法对比

| 特点     | K-Means    | 层次聚类 | DBSCAN           |
| -------- | ---------- | -------- | ---------------- |
| 簇数     | 需提前指定 | 无需     | 无需             |
| 形状     | 球形       | 任意     | 任意             |
| 速度     | 快         | 慢       | 中等             |
| 异常值   | 敏感       | 敏感     | 鲁棒             |
| 可解释性 | 强         | 强       | 强               |
| 超参数   | K          | 链接方法 | eps, min_samples |

## 7. 完整工作流程

1. **数据准备**

   - 加载数据
   - 处理缺失值
   - 特征缩放（重要！）
2. **选择算法**

   - 簇是否凸？ → K-Means
   - 簇有不同形状或密度？ → DBSCAN或层次聚类
   - 需要簇树？ → 层次聚类
3. **参数选择**

   - K-Means：肘部法则或轮廓系数
   - DBSCAN：k距离图
   - 层次：树状图
4. **训练模型**
5. **评估聚类**

   - 轮廓系数
   - 惯性
   - 可视化
6. **解释结果**

   - 簇的特征
   - 业务意义
