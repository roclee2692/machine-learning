### 背景

### 数据生成
```python
from sklearn.datasets import make_classification

X, y = make_classification(
    n_samples=10000,
    n_features=20,
    n_informative=15,
    n_redundant=5,
    weights=[0.95, 0.05],  # 95%正常, 5%欺诈

)
```

### 任务
1. **数据探索**
   - 查看类别分布

   - 不能只看准确率（为什么？）

   - 绘制ROC曲线和PR曲线

   - 找到最优决策阈值
   - 平衡精确率和召回率

```python

data = {
    'age': [25, 35, 45, 55, 65],
    'income': [30000, 50000, 70000, 90000, 110000],
    'tenure_months': [6, 24, 48, 72, 96],
```
