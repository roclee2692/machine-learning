## 🔑 核心学习要点总结

### 逻辑回归（Logistic Regression）
✓ **是什么**: 区分性的概率分类模型
✓ **学什么**: 决策边界 P(y|x) = σ(w·x + b)
✓ **怎么学**: 最小化交叉熵损失函数
✓ **优点**: 
  - 准确率通常很高
  - 无需分布假设
  - 快速高效
✓ **缺点**: 
  - 小数据集容易过拟合
  - 需要线性可分性较好

### 朴素贝叶斯（Naive Bayes）
✓ **是什么**: 生成性的概率分类模型
✓ **学什么**: 数据分布 P(x|y) 和先验 P(y)
✓ **怎么学**: 最大化似然函数
✓ **优点**: 
  - 稳定性好，对噪声鲁棒
  - 小数据集表现稳定
  - 可以生成新数据
✓ **缺点**: 
  - 独立性假设不现实
  - 准确率通常略低于区分性模型

### 区分性 vs 生成性的本质区别
✓ **区分性**: 直接学 "怎么分" → P(y|x)
✓ **生成性**: 先学 "什么样子" → P(x|y)、P(y)，再推导 P(y|x)

## 🎯 生成的可视化图表总览

### 决策边界图（最重要！）
- 展示模型如何分割特征空间
- 可直观看到区分性 vs 生成性的区别
- 通常决策边界是线性 vs 二次曲线

### 混淆矩阵（评估模型）
- 显示模型在各类别上的表现
- 可以看出模型偏向哪个类别
- 计算精确率、召回率等指标的基础

### 概率分布图（理解模型）
- 展示模型对预测概率的分布
- 看出模型的不确定性
- 可用于调整决策阈值

### 热力图（可视化概率）
- 在二维特征空间中展示概率
- 颜色深浅表示概率大小
- 特别有用的可视化工具
