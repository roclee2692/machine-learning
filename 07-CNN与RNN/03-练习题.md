```

```

## 答案提示

### 练习 1 答案
```
-1  -2  -1
-1  -2  -1
-1  -2  -1
```
这个卷积核检测竖直边缘，输出负值表示"左边亮、右边暗"的竖直边缘。

### 练习 2 答案
1. 遗忘门接近 0 → 旧的细胞状态被清空，相当于"重置记忆"
2. 输入门接近 0 → 不写入任何新信息，保持旧记忆不变
3. 输入门（Input Gate）值为 1 时，完全写入新的候选值

### 练习 5 答案
1. `nn.Conv1d` 输入形状：`(batch, channels, length)`，如 `(128, 1, 100)` 表示 128 个样本，每个是长度 100 的序列
2. 1D 卷积核在扫描连续的几个词（如 2-gram、3-gram），提取局部语义特征
3. TextCNN 优势：并行计算快、能捕捉局部 n-gram 特征；LSTM 优势：能捕捉长距离依赖、处理变长序列
