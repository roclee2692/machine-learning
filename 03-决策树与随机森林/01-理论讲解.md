# 决策树与随机森林 (Decision Tree & Random Forest)

## 1. 什么是决策树？

决策树是一种简单而强大的**分类和回归**算法，通过一系列的"是/否"问题来分割数据。

### 1.1 直观理解

想象你要判断一个人是否容易患心脏病：
```
                    年龄 > 50？
                   /          \
                 是             否
                /                \
        血压 > 140？         运动量 > 3次/周？
        /      \            /          \
       是      否          是           否
      患病    正常        正常        患病
```

### 1.2 树的基本概念

- **根节点**：最顶部的决策点
- **内部节点**：其他决策点
- **叶子节点**：最终的类别预测
- **分支**：yes/no的选择

## 2. 数学原理

### 2.1 如何选择最佳分割？

关键是找到能最好地分离类别的特征和阈值。常用两种度量方式：

**方法1：信息增益 (Information Gain)**

```
信息增益 = 分割前的熵 - 分割后的熵

Entropy(S) = -Σ p_i * log₂(p_i)

其中 p_i 是类别i的比例
```

**方法2：基尼不纯度 (Gini Impurity)**

```
Gini(S) = 1 - Σ p_i²

信息增益 = Gini(父) - Σ(n_子/n_父) * Gini(子)
```

**哪个更好？**
- 信息增益：易于理解，计算量大
- 基尼指数：计算更快，sklearn默认使用

### 2.2 何时停止分割？

- 叶子节点中所有样本都属于同一类别
- 达到最大深度（防止过拟合）
- 节点中样本数少于最小值
- 无法找到更好的分割

## 3. 超参数

### 3.1 重要超参数

**max_depth**：树的最大深度
- 限制树的复杂度
- 默认None（无限深）
- 增加depth会增加过拟合风险

**min_samples_split**：分割一个节点所需的最小样本数
- 默认2
- 增大可以减少过拟合

**min_samples_leaf**：叶子节点的最小样本数
- 默认1
- 防止过拟合

**max_features**：选择分割特征时考虑的最大特征数
- "sqrt"：√n个特征
- "log2"：log₂(n)个特征
- None：所有特征

## 4. 决策树的优缺点

### 优点
- ✅ 易于理解和解释
- ✅ 支持分类和回归
- ✅ 无需特征缩放
- ✅ 可视化能力强
- ✅ 自动特征交互

### 缺点
- ❌ 容易过拟合
- ❌ 对数据微小变化敏感
- ❌ 不稳定
- ❌ 对不平衡数据敏感

## 5. 什么是随机森林？

**随机森林**是多个决策树的组合体，通过**投票**的方式做出最终决策。

### 5.1 如何构建随机森林？

1. 从原始数据集中，用**放回抽样**创建多个子数据集
2. 对每个子数据集，训练一个独立的决策树
3. 每棵树可以独立地达到最大深度（完全生长）
4. **融合**：
   - 分类：多数投票
   - 回归：平均值

### 5.2 随机性来自哪里？

- **样本随机**：Bagging（自助聚合）
- **特征随机**：每次分割只考虑部分特征

这种**"有组织的随机"**产生了强大的泛化能力。

## 6. 随机森林的优缺点

### 优点
- ✅ 减少过拟合
- ✅ 处理大规模数据快速
- ✅ 提供特征重要性排序
- ✅ 健壮性好
- ✅ 并行化容易

### 缺点
- ❌ 模型复杂，不易解释
- ❌ 内存占用大
- ❌ 对噪声敏感

## 7. 特征重要性

### 7.1 如何计算？

在每棵树中，特征的重要性 = 该特征导致的信息增益之和

```
特征重要性 = Σ(各棵树中该特征的信息增益)
```

### 7.2 应用

- 特征选择
- 理解模型决策
- 识别关键变量

## 8. 过拟合与欠拟合

### 8.1 识别问题

```
训练准确率    验证准确率    问题
95%          60%          过拟合（树太深）
60%          55%          欠拟合（树太浅）
85%          84%          刚好
```

### 8.2 解决方案

**过拟合**：
- 降低max_depth
- 增加min_samples_split
- 增加min_samples_leaf

**欠拟合**：
- 增加max_depth
- 减少min_samples_split
- 减少min_samples_leaf

## 9. 决策树 vs 随机森林

| 特点 | 决策树 | 随机森林 |
|------|--------|---------|
| 准确率 | 中等 | 高 |
| 过拟合 | 严重 | 轻微 |
| 可解释性 | 强 | 弱 |
| 速度 | 快 | 中等 |
| 特征重要性 | 可得 | 可得 |

## 10. 完整工作流程

1. **数据准备**
   - 加载数据
   - 检查缺失值
   - 处理分类特征

2. **特征工程**
   - 特征选择
   - 编码分类变量

3. **划分数据集**
   - 训练集、验证集、测试集

4. **训练模型**
   - 决策树or随机森林
   - 设置超参数

5. **评估模型**
   - 准确率、精确率、召回率
   - 混淆矩阵、ROC曲线
   - 特征重要性

6. **可视化树**
   - 查看决策过程

7. **优化模型**
   - 调整超参数
   - GridSearchCV
