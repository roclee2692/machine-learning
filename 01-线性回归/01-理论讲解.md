# 线性回归 (Linear Regression)

## 1. 什么是线性回归？

线性回归是最基础的机器学习算法，用于预测**连续值**。它假设特征和目标值之间存在线性关系。

### 1.1 直观理解

想象你要预测房价：

- **输入**：房屋面积 (x)
- **输出**：房价 (y)

如果发现面积越大，房价越高，且大致成正比，那就可以用一条直线来拟合这个关系：

```
y = wx + b
```

- `w` (weight/权重)：直线的斜率
- `b` (bias/偏置)：直线的截距

### 1.2 多元线性回归

实际问题中通常有多个特征：

```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
```

向量形式：

```
y = w^T x + b
```

其中：

- `x = [x₁, x₂, ..., xₙ]^T` 是特征向量
- `w = [w₁, w₂, ..., wₙ]^T` 是权重向量

─────────────────────────────
样本数多，特征少    → Linear Regression
                  （直接用）

                  （所有特征都用，但权重降低）

* **Linear** ：快速但容易过拟合
* **Ridge** ：平衡解，保留所有特征但缩小权重
* **Lasso** ：做特征筛选，得到稀疏模型

## 2. 数学原理

### 2.1 假设函数 (Hypothesis)

```
h(x) = w^T x + b
```

简化表示（把b合并到w中）：

```
h(x) = w^T x
```

### 2.2 损失函数 (Cost Function)

我们需要找到最好的w和b，使得预测值h(x)和真实值y之间的误差最小。

**均方误差 (MSE - Mean Squared Error)**：

```
J(w,b) = 1/(2m) Σᵢ₌₁ᵐ (h(xⁱ) - yⁱ)²
```

其中：

- m 是训练样本数量
- (xⁱ, yⁱ) 是第i个训练样本
- 1/2 是为了求导方便

### 2.3 梯度下降 (Gradient Descent)

目标：找到使 J(w,b) 最小的参数w和b

**更新规则**：

```
w := w - α ∂J/∂w
b := b - α ∂J/∂b
```

其中 α 是学习率 (learning rate)

**梯度计算**：

```
∂J/∂w = 1/m Σᵢ₌₁ᵐ (h(xⁱ) - yⁱ) · xⁱ
∂J/∂b = 1/m Σᵢ₌₁ᵐ (h(xⁱ) - yⁱ)
```

### 2.4 正规方程 (Normal Equation)

除了梯度下降，线性回归还有解析解（闭式解）：

```
w = (X^T X)⁻¹ X^T y
```

**梯度下降 vs 正规方程**：

| 特点   | 梯度下降         | 正规方程            |
| ------ | ---------------- | ------------------- |
| 速度   | 特征多时仍然快   | 特征多时很慢 O(n³) |
| 适用性 | 适用于大规模数据 | n < 10000时可用     |
| 调参   | 需要选择α       | 不需要              |
| 迭代   | 需要多次迭代     | 一次计算            |

## 3. 评估指标

### 3.1 均方误差 (MSE)

```
MSE = 1/m Σᵢ₌₁ᵐ (yᵢ - ŷᵢ)²
```

### 3.2 均方根误差 (RMSE)

```
RMSE = √MSE
```

### 3.3 平均绝对误差 (MAE)

```

Mean Absolute Error
缩写：MAE

中文：平均绝对误差
MAE = 1/m Σᵢ₌₁ᵐ |yᵢ - ŷᵢ|
```

### 3.4 R² 决定系数

```
R² = 1 - SS_res/SS_tot
```

其中：

- SS_res = Σ(yᵢ - ŷᵢ)² (残差平方和)
- SS_tot = Σ(yᵢ - ȳ)² (总平方和)

**R² 解释**：

- R² = 1：完美拟合
- R² = 0：模型和预测均值一样
- R² < 0：模型比预测均值还差

R² 公式的逻辑：

1. SS_tot  = 数据本身有多"波动"（不用模型时的误差）
2. SS_res  = 用模型后还剩多少"误差"
3. SS_res/SS_tot = 模型没解释的比例
4. 1 - (没解释的比例) = 模型解释的比例 = R²

### 4.1 特征缩放 (Feature Scaling)

当特征的尺度差异很大时，梯度下降会很慢。加快收敛速度

**j归一化 (Min-Max Scaling)**：

```
x' = (x - x_min) / (x_max - x_min)
```

将特征缩放到 [0, 1]

**标准化 (Standardization)**：

```
x' = (x - μ) / σ
```

将特征转换为均值0、方差1的分布

### 4.2 多项式回归

如果数据不是线性的，可以添加多项式特征：

```
原始特征: x
添加特征: x, x², x³, ...
```

这样仍然是线性回归（对参数w是线性的），但可以拟合非线性关系。

## 5. 过拟合与欠拟合

### 5.1 欠拟合 (Underfitting)

- **原因**：模型太简单
- **表现**：训练误差大，测试误差也大
- **解决**：增加特征、增加多项式次数

### 5.2 过拟合 (Overfitting)

- **原因**：模型太复杂，学习了噪声
- **表现**：训练误差很小，测试误差大
- **解决**：
  - 获取更多训练数据
  - 减少特征
  - 正则化

### 5.3 正则化 (Regularization)

**L2 正则化 (Ridge Regression)**：

```
J(w) = 1/(2m) Σᵢ₌₁ᵐ (h(xⁱ) - yⁱ)² + λ/(2m) Σⱼ₌₁ⁿ wⱼ²
```

**L1 正则化 (Lasso Regression)**：

```
J(w) = 1/(2m) Σᵢ₌₁ᵐ (h(xⁱ) - yⁱ)² + λ/m Σⱼ₌₁ⁿ |wⱼ|
```

其中 λ 是正则化参数：

- λ = 0：无正则化
- λ 很大：强正则化（可能导致欠拟合）

## 7. 常见问题

### Q1: 什么时候用线性回归？

- 预测连续值
- 特征和目标大致呈线性关系
- 需要模型可解释性

### Q2: 学习率怎么选？

- 太小：收敛慢
- 太大：可能不收敛
- 典型值：0.001, 0.01, 0.1, 1
- 可以画损失函数曲线来调整

### Q3: 如何判断是否收敛？

- 损失函数不再明显下降
- 达到最大迭代次数
- 梯度接近0

### Q4: 多重共线性怎么办？

当特征之间高度相关时：

- 删除相关特征
- 使用Ridge回归
- 使用主成分分析(PCA)降维
