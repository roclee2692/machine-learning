# 手写线性回归算法 - 完整学习指南

## 📊 核心概念

线性回归的目标是找到一条最佳拟合直线，使得预测值尽可能接近真实值。

```
模型方程: ŷ = w*x + b
其中：
  ŷ = 预测值
  w = 权重（直线的斜率）
  b = 偏置（直线的截距）
  x = 输入特征
```

---

## 🎯 4个关键步骤

### **步骤1：前向传播（Forward Propagation）**

使用当前的权重和偏置计算预测值。

```python
def forward(X, w, b):
    """前向传播"""
    y_pred = np.dot(X, w) + b  # ŷ = X*w + b
    return y_pred
```

**示例**：
- 输入特征：X = [50, 75, 100]（房屋面积）
- 权重：w = 3.5，偏置：b = 50
- 预测：ŷ = [50, 75, 100] * 3.5 + 50 = [225, 312.5, 400]

---

### **步骤2：计算损失函数（Loss Function）**

均方误差（MSE）是最常用的损失函数，衡量预测值与真实值的偏差。

```python
def compute_loss(y_pred, y_true):
    """计算均方误差"""
    m = len(y_true)
    mse = (1/m) * np.sum((y_pred - y_true) ** 2)
    return mse
```

**数学公式**：
$$L = \frac{1}{m} \sum_{i=1}^{m} (ŷ_i - y_i)^2$$

**说明**：
- $m$ = 样本数量
- 损失函数越小，模型效果越好
- 损失值应该逐步下降

**示例**：
```
真实值：y = [220, 310, 395]
预测值：ŷ = [225, 312.5, 400]
误差：[5, 2.5, 5]
MSE = (1/3) * (25 + 6.25 + 25) = 18.75
```

---

### **步骤3：计算梯度（Gradient Computation）**

梯度告诉我们参数应该往哪个方向调整。

```python
def compute_gradients(X, y_pred, y_true):
    """计算梯度"""
    m = len(y_true)
    error = y_pred - y_true
    
    # w的梯度
    dw = (2/m) * np.dot(X.T, error)
    
    # b的梯度
    db = (2/m) * np.sum(error)
    
    return dw, db
```

**数学公式**：
$$\frac{\partial L}{\partial w} = \frac{2}{m} \sum_{i=1}^{m} (ŷ_i - y_i) \cdot x_i$$

$$\frac{\partial L}{\partial b} = \frac{2}{m} \sum_{i=1}^{m} (ŷ_i - y_i)$$

**含义解释**：
- 梯度是损失函数关于参数的偏导数
- 梯度告诉我们函数的"斜率"
- 沿着负梯度方向走，就能找到最小值

---

### **步骤4：参数更新（Gradient Descent）**

根据梯度和学习率更新参数。

```python
def update_parameters(w, b, dw, db, learning_rate):
    """梯度下降更新参数"""
    w = w - learning_rate * dw
    b = b - learning_rate * db
    return w, b
```

**更新公式**：
$$w_{new} = w_{old} - \alpha \cdot \frac{\partial L}{\partial w}$$

$$b_{new} = b_{old} - \alpha \cdot \frac{\partial L}{\partial b}$$

**参数说明**：
- $\alpha$ = 学习率（learning rate），控制每次更新的步长
- 学习率太大：参数会波动，可能发散
- 学习率太小：收敛很慢

## 🔄 训练过程（完整流程）

```
初始化参数 (w=0, b=0)
    ↓
重复以下步骤N次 (或直到收敛):
    1️⃣  前向传播: 计算预测值 ŷ = X*w + b
    2️⃣  计算损失: L = (1/m) * Σ(ŷ - y)²
    3️⃣  反向传播: 计算梯度 dw, db
    4️⃣  参数更新: w -= α*dw, b -= α*db
    5️⃣  记录损失值
    ↓
输出最终参数 w, b
```

## 💡 重要概念详解

### **1. 为什么要归一化？**

```python
# 归一化：将特征缩放到 [0, 1]
X_normalized = (X - X.min()) / (X.max() - X.min())
```

**原因**：
- 特征值范围差异大时，梯度计算容易出现数值不稳定
- 未经归一化的数据会导致权重更新过大或过小
- 归一化后，学习率更容易调整

### **2. 学习率的影响**

| 学习率 | 结果 |
|--------|------|
| **太大** (如 0.1) | 损失函数波动，甚至发散（NaN） |
| **合适** (如 0.01) | 损失函数平稳下降 |
| **太小** (如 0.0001) | 收敛太慢，需要很多迭代 |

### **3. R² 分数的含义**

```
R² = 1 - (SS_res / SS_tot)
```

- R² = 1：完美预测
- R² = 0：与平均值一样差
- R² < 0：模型比平均线更差

在示例中：
- 训练集 R² = 0.886735（约89%的方差被解释）
- 测试集 R² = 0.880654

## 🔧 完整代码框架

### **最小化实现（简洁版）**

```python
import numpy as np

class SimpleLinearRegression:
    def __init__(self, learning_rate=0.01, iterations=100):
        self.lr = learning_rate
        self.iters = iterations
        self.w = None
        self.b = None
    
    def fit(self, X, y):
        m, n = X.shape
        self.w = np.zeros(n)
        self.b = 0
        
        for _ in range(self.iters):
            # 前向传播
            y_pred = np.dot(X, self.w) + self.b
            
            # 计算梯度
            error = y_pred - y
            dw = (2/m) * np.dot(X.T, error)
            db = (2/m) * np.sum(error)
            
            # 参数更新
            self.w -= self.lr * dw
            self.b -= self.lr * db
        
        return self
    
    def predict(self, X):
        return np.dot(X, self.w) + self.b
```

## 📝 常见问题

### **Q1: 为什么权重初始化为0？**
- 初始化为0可以保证公平性
- 权重会根据数据自动学习到合适的值
- 其他初始化方法（如随机初始化）也可以，但需要更多调整

### **Q2: 什么时候停止训练？**
选项：
1. **固定迭代次数**：如 500 次迭代
2. **早停法**：当验证集损失不再下降时停止
3. **收敛判断**：当损失变化小于阈值时停止

### **Q3: 如何处理多维特征？**
- 代码自动支持！`np.dot(X, w)` 会处理 X 的所有特征列
- 每个特征都有对应的权重 w

### **Q4: 线性回归的局限性？**
- 只能拟合线性关系
- 如果数据非线性，需要添加多项式特征或使用其他模型
